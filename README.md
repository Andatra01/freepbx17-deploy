# FreePBX 17 Infrastructure as Code

Проект для автоматизированного развёртывания FreePBX 17 на Debian 12 с использованием:
- Ansible (управление конфигурацией)
- Terraform (Yandex Cloud в перспективе)
- Git (контроль версий)
- IN1CLICK installer (улучшенный официальный скрипт)

## Структура
- `ansible/` - плейбуки и роли Ansible
- `terraform/` - конфигурации Terraform для облака
- `scripts/` - вспомогательные скрипты
- `docs/` - документация
